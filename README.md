# Latex Styles and Classes Library

This repository contains custom `.cls` and `.sty` files for my own LaTeX documents.

## Requirements
- TeX Live (2024 or later recommended)
- LuaLaTeX
- latexmk

## Usage 

```bash
latexmk -lualatex sample.tex
```


## Notes
- These styles are designed for personal use but may be useful for others.
- Tested with TeX Live 2025 on macOS.
